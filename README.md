
  # PSYLEARN Quiz App

  This is a code bundle for PSYLEARN Quiz App. The original project is available at https://www.figma.com/design/RNDneqN22w9fFPLrC79fBG/PSYLEARN-Quiz-App.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  
## Mobile App / PWA Support

This project is now prepared for mobile use as a PWA. See `MOBILE_APP_SETUP.md` for Android and iPhone installation steps, plus optional Capacitor packaging steps.
